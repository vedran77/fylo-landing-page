# flyo-landing-page  

## Technology used:
### VueJS - 3
### TypeScript - 4.5.0
### Tailwind CSS - 3.0.13

## Live Preview
[GitHub Pages]https://vedran77.github.io/fylo-landing-page/

### Setup flyo-landing-page
```
git clone git@github.com:vedran77/fylo-landing-page.git
npm install
```

### For production 
```
npm run build
```

### For development
```npm run serve```
